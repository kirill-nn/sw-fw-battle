# **Известные недостатки и сомнения по решению**
1. нет отслеживания доступности клетки для хода (необходимо расширить логику класса arena), на данном этапе необходимости нет, поскольку юниты вступают в схатку при сближении
2. метод tick() в симуляторе выглядит громоздким, возможно есть смысл декомпозировать
3. нет валидации соответствия команд созданным юнитам, т.е. если для команды не найден юнит по ID - команда пропускается
4. нет валидации уникальности ID при создании юнитов
